# Memos Al Sync

原作者仓库为：[leoleelxh/obsidian-memos-ai-sync: obsidian-memos-sync-plugin，将 Memos 内容同步到 Obsidian 的插件，提供无缝集成体验。](https://github.com/leoleelxh/obsidian-memos-ai-sync)

因为不支持memos 0.25之后的版本API，同步后只同步文字，不同步图片，而且作者似乎断更了，所以我临时用AI更新了一个支持0.25之后版本的，大家不嫌弃的话，请自取。

只需下载项目中的main.js文件，然后覆盖到obsidian知识库的.obsidian\plugins\memos-ai-sync文件夹中即可。
